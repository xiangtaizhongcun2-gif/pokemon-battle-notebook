# Pokémon Battle Notebook

ポケモンのパーティ構築と対戦の振り返りを、ひとつの場所で管理するシンプルなWebアプリです。

## 主な機能

- パーティの作成・編集・削除
- 1パーティにつき最大6匹の登録
- ポケモン図鑑の検索
- タイプ、弱点、耐性、無効、種族値の表示
- 特性、持ち物、性格、テラスタイプ、技4つの記録
- 努力値の入力と合計510チェック
- 構築コンセプト、基本選出、苦手な相手のメモ
- 対戦履歴の追加・編集・削除
- LocalStorageへの自動保存

## UI方針

画面は次の3つだけです。

1. パーティ
2. ポケモン図鑑
3. 対戦履歴

白背景、黒とグレーを中心に、装飾や画面遷移を最小限にしています。Notionの複雑なページ機能は採用せず、余白と読みやすさだけを参考にしています。

## 現在の図鑑データ

最初のバージョンでは、対戦でよく使われるポケモンを中心にローカルデータとして収録しています。図鑑データは `src/simple/data.ts` の `POKEDEX` へ追加できます。

タイプ相性は特性による変更を考慮せず、ポケモン本来のタイプから計算します。

## 開発環境

- Vite
- React
- TypeScript
- LocalStorage

## 起動方法

```bash
npm install
npm run dev
```

## 確認コマンド

```bash
npm run build
npm run lint
npm run format:check
```

## 対象外

最初のバージョンでは、次の機能は実装しません。

- ダメージ計算
- ログイン
- クラウド同期
- パーティ共有
- ターン単位の詳細な対戦ログ
- Notion風ブロックエディター
